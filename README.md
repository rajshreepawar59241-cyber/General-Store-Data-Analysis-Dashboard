# General Store Data Analysis(Interactive Dashboard using MS Excel)
## Project Overview
This project focuses on analyzing general store sales data to understand customer behavior, sales performance, order status, and channel contribution.
The goal is to derive actionable business insights that can help improve sales strategy and customer targeting.
