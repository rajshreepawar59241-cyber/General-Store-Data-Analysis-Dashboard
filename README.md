# General Store Data Analysis(Interactive Dashboard using MS Excel)
## Project Overview
This project focuses on analyzing general store sales data to understand customer behavior, sales performance, order status, and channel contribution.
The goal is to derive actionable business insights that can help improve sales strategy and customer targeting.

## Dataset used
- <a href="https://github.com/rajshreepawar59241-cyber/General-Store-Data-Analysis-Dashboard/blob/0b152cb9568b3c44347776f1dedad626e492adb3/genral%20store%20data%20set.xlsx">Dataset</a>

## process
Analyze sales trends across gender, age group, state, and sales channels
Identify top-performing customer segments
Understand order fulfillment performance
Provide data-driven recommendations to improve store sales

Dashboard Interaction <a href="https://github.com/rajshreepawar59241-cyber/General-Store-Data-Analysis-Dashboard/blob/0b152cb9568b3c44347776f1dedad626e492adb3/general_store_db.png">view Dashboard</a>

## Key Insights:
✅ Women contribute ~64% of total sales
✅ Adult age group (30–49 years) is the highest revenue contributor
✅ Maharashtra, Karnataka & Uttar Pradesh are the top-performing states
✅ Amazon, Flipkart & Myntra drive maximum sales
✅ 92% of orders are successfully delivered, showing strong operations
